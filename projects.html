Generate new music

Generating music is formulated as a next-note prediction problem that involves using Recurrent Neural Networks.. In this project, we have used Music21 and Keras to build the LSTM model for generating music. We have also created model architecture, checkpoints, and loss functions and learnt to predict notes using random input. The main goal has been to use MIDI files to train neural networks, extract output from the model, and convert them into MP3 music file.

Cybercrime is on the rise and cybercriminals are using various techniques to hide their digital footprint and evade detection. This project builds a solution that can determine if an IP address is a proxy or VPN IP address and trace the actual IP address behind it.
